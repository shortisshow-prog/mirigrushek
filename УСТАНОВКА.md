# Инструкция по установке сервера «МирИгрушек»

Пошаговое руководство: что делать и какие данные вводить при настройке сервера
с нуля и развёртывании информационной системы магазина.

---

## 0. Что понадобится

- Виртуальная машина или сервер с **Ubuntu Server 22.04 / 24.04** (подойдёт и Desktop).
- Доступ в интернет на сервере (для загрузки пакетов).
- На вашем компьютере: браузер и **MySQL Workbench**.

---

## 1. Установка Ubuntu (что вводить в мастере установки)

Если Ubuntu уже установлен — переходите к шагу 2.

| Экран мастера | Что выбрать / ввести |
|---|---|
| **Language** | English (или Русский) |
| **Keyboard** | оставить по умолчанию (English US) |
| **Type of install** | Ubuntu Server (обычная установка) |
| **Network** | оставить автоматически (DHCP). Запишите выданный **IP-адрес** — он понадобится |
| **Proxy address** | оставить **пустым** (просто Enter) |
| **Mirror address** | оставить по умолчанию |
| **Storage / диск** | Use an entire disk → Done → Continue (подтвердить запись на диск) |
| **Profile setup** | заполнить: |
| &nbsp;&nbsp;• Your name | например `admin` |
| &nbsp;&nbsp;• Your server's name | например `mirigrushek` |
| &nbsp;&nbsp;• Pick a username | например `admin` |
| &nbsp;&nbsp;• Password | **придумайте и запишите** (это пароль для входа в Ubuntu) |
| **SSH Setup** | поставьте галочку **Install OpenSSH server** (удобно подключаться удалённо) |
| **Featured snaps** | ничего не отмечать → Done |

Дождитесь окончания установки → **Reboot Now**. Войдите под созданными
**username/паролем** из шага «Profile setup».

---

## 2. Узнать IP-адрес сервера

После входа выполните:

```bash
ip a
```

или короче:

```bash
hostname -I
```

Запишите адрес вида `192.168.x.x` — по нему будете открывать сайт и подключаться в Workbench.

---

## 3. Установка системы одной командой

Выполните на сервере (потребуется ввести пароль пользователя для `sudo`):

```bash
curl -fsSL https://raw.githubusercontent.com/shortisshow-prog/mirigrushek/main/install.sh | sudo bash
```

> **Никаких данных вводить во время установки не нужно** — скрипт работает автоматически:
> ставит Apache + MySQL + PHP, создаёт базу данных, настраивает доступ и разворачивает сайт.
> Установка занимает 2–5 минут.

Когда всё закончится, скрипт напечатает примерно такое:

```
============================================================
  ГОТОВО.
  Сайт:               http://192.168.x.x/
  MySQL Workbench:    host=192.168.x.x  port=3306  user=root  pass=Xmpl123!
  База данных:        mirigrushek
  Пользователи сайта: логин = email из задания, пароль = Xmpl123!
============================================================
```

---

## 4. Открыть сайт

В браузере на вашем компьютере откройте:

```
http://<IP-адрес-сервера>/
```

Например `http://192.168.1.50/`. Должен сразу открыться каталог товаров
(без приветственной страницы Apache).

---

## 5. Подключение в MySQL Workbench

1. Откройте MySQL Workbench → нажмите **➕** рядом с «MySQL Connections».
2. Введите параметры:

   | Поле | Значение |
   |---|---|
   | **Connection Name** | `mirigrushek` (любое) |
   | **Hostname** | IP-адрес сервера (например `192.168.1.50`) |
   | **Port** | `3306` |
   | **Username** | `root` |

3. Нажмите **Store in Vault / Set Password** и введите пароль: **`Xmpl123!`**
4. **Test Connection** → должно появиться «Successfully made the MySQL connection».
5. **OK** → откройте соединение. Слева в **SCHEMAS** будет база **`mirigrushek`**.

## 7. Если что-то не работает

| Проблема | Решение |
|---|---|
| `curl: command not found` | `sudo apt update && sudo apt install -y curl`, затем повторить команду из шага 3 |
| Сайт не открывается в браузере | проверьте IP (`hostname -I`); убедитесь, что компьютер и сервер в одной сети |
| Workbench не подключается | на сервере: `sudo systemctl status mysql` (должно быть `active`); проверьте, что вводите порт `3306` и пароль `Xmpl123!` |
| Нужно переустановить с нуля | просто запустите команду из шага 3 ещё раз — она пересоздаёт базу и сайт |
| Открывается стандартная страница Apache | выполните `sudo rm -f /var/www/html/index.html && sudo systemctl reload apache2` |

## Памятка по доступам

- **Вход в Ubuntu:** username/пароль, заданные при установке (шаг 1).
- **MySQL root:** пароль `Xmpl123!`, подключение с любого хоста, порт `3306`.
- **Вход на сайт:** фамилия латиницей + `@mail.ru` (например `vorsin@mail.ru`) + пароль `Xmpl123!`. Гость может зарегистрироваться.
