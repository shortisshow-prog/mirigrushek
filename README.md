# ООО «МирИгрушек» — ИС магазина игрушек (ДЭ 09.02.07-2-2026, вариант В4)

Полный пакет для развёртывания на чистом **Ubuntu**-сервере (LAMP) одной командой.

## Установка на сервер одной командой

На чистом Ubuntu-сервере выполните:

```bash
curl -fsSL https://raw.githubusercontent.com/shortisshow-prog/mirigrushek/main/install.sh | sudo bash
```

Скрипт сам поставит git, склонирует репозиторий в `/opt/mirigrushek` и запустит установку
(Apache + MySQL + PHP, создание БД, развёртывание сайта).

## Развёртывание вручную (если репозиторий уже скачан)

Скопируйте папку `mirigrushek` на сервер и выполните **одну команду** из неё:

```bash
sudo bash setup.sh
```

Скрипт:
1. ставит Apache, MySQL, PHP;
2. создаёт БД `mirigrushek` со всеми таблицами и связями (`init.sql`);
3. делает пользователя **root доступным отовсюду** с паролем `Xmpl123!` (для MySQL Workbench), `bind-address = 0.0.0.0`;
4. разворачивает сайт и **убирает приветственную страницу Apache** — по адресу сервера сразу открывается каталог товаров.

По завершении скрипт печатает адрес сайта и параметры подключения.

## Доступы

| Что | Значение |
|---|---|
| Сайт | `http://<IP-сервера>/` |
| MySQL Workbench | host = `<IP>`, port = `3306`, user = `root`, pass = `Xmpl123!` |
| База данных | `mirigrushek` |
| Вход в приложение | логин = e-mail из задания, пароль = `Xmpl123!` (у всех) |

## Роли (по заданию)
- **Гость** (без входа) — только просмотр каталога.
- **Авторизированный клиент** — просмотр каталога (без фильтра/сортировки/поиска).
- **Менеджер** — каталог с фильтром/сортировкой/поиском + просмотр заказов.
- **Администратор** — всё + добавление/редактирование/удаление товаров и заказов.

## Структура БД

Справочники: `Roles`, `Categories`, `Suppliers`, `Manufacturers`, `Units`, `OrderStatuses`, `PickupPoints`.
Основные: `Users` (→Roles), `Products` (→Units/Suppliers/Manufacturers/Categories),
`Orders` (→PickupPoints/Users/OrderStatuses), `OrderItems` (→Orders/Products) — состав заказа «многие-ко-многим».

## Реализованные правила оформления (Руководство по стилю М2)
- шрифт Arial, заголовок на каждой форме, логотип/иконка из ресурсов;
- фон `#FFFFFF`, доп. фон `#F5DEB3`, акцент `#DEB887`;
- скидка **> 17 %** → фон карточки `#FFDEAD`;
- товара нет на складе (кол-во = 0) → цена красным;
- нет фото → заглушка `picture.png`;
- загрузка фото — минимум **300×200 px**, цена и количество ≥ 0.

## Очистка данных (М1)
Заказ №7 в исходном файле содержал несуществующую дату `30.02.2025` — при импорте она
сохранена как `NULL` (некорректные даты не загружаются).
